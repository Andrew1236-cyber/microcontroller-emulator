# Эмулятор Микроконтроллера

## Описание
Простой эмулятор 8-битного микроконтроллера с собственной системой команд, написанный на Python.

## Архитектура
- 8-битная архитектура
- 16 регистров общего назначения (R0-R15)
- 64KB адресного пространства
- 16 инструкций

## Система команд

| Инструкция | Описание | Код |
|------------|----------|-----|
| NOP        | Нет операции | 0x00 |
| LDI Rd, imm | Загрузка непосредственного значения | 0x01 |
| MOV Rd, Rs | Копирование регистров | 0x02 |
| ADD Rd, Rs | Сложение | 0x03 |
| SUB Rd, Rs | Вычитание | 0x04 |
| JMP addr   | Безусловный переход | 0x05 |
| JZ addr    | Переход если Z=1 | 0x06 |
| JNZ addr   | Переход если Z=0 | 0x07 |
| CALL addr  | Вызов подпрограммы | 0x08 |
| RET        | Возврат из подпрограммы | 0x09 |
| PUSH Rs    | Поместить в стек | 0x0A |
| POP Rd     | Извлечь из стека | 0x0B |
| AND Rd, Rs | Побитовое И | 0x0C |
| OR Rd, Rs  | Побитовое ИЛИ | 0x0D |
| XOR Rd, Rs | Исключающее ИЛИ | 0x0E |
| HALT       | Остановка | 0x0F |
